# HOW TO JOIN

## Step 1 – Join Discord
https://discord.gg/fWXqjJDRsA

## Step 2 – Direct Connect (Arma 3)

1. Launch Arma 3 Launcher  
2. Navigate to Servers  
3. Select Direct Connect  
4. Enter Server IP + Port  
5. Connect  

If you receive an error, you likely need required mods.

---

## Step 3 – Install Required Mods

Once you connect:

- Exit game
- Go to Launcher → Recent
- Select server
- Review required mods
- Subscribe via Steam Workshop

We recommend building a preset in the launcher and saving it.

You can import the provided modlist file and let the launcher auto-download required content.

# CAS DOCTRINE

All pilots must acknowledge rules before flying CAS.

## CAS Engagement Rules

- Verify targets using map before firing
- Do not bomb HQ or mortars
- Avoid striking close to FOB
- Minimum strike distances:
  - 75m missile/rocket strikes
  - 100m bombs (except napalm)
  - 150m napalm / daisy cutter
- Leave custom objectives to ground troops on event days
- Deliver supplies when not conducting air support
- Never leave men on ground during extract
- Maintain safe distance between aircraft

Primary mission:
Support ground troops — not destroy everything.

---

# RTO GUIDE

Supports may be called once per restart. Some recharge.

Strike timing:

- 15 seconds between requests
- ~1 minute until impact
- 25 seconds from “aircraft inbound” to impact

Examples:

- Commando Vault (Daisy Cutter) – 30 min cooldown  
- Arc Light – 30 min  
- F4 Bomb Strike – 5.5 min  
- Napalm Strike – 5.5 min  
- 20mm Gun Run – 2.5 min  

Illumination – 5 min recharge.
